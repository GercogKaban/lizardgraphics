Detailed Description

Компонент необходимы для конструкции виджетов Lizard Graphics.

Сожержит в себе шейдер и буфер вершин, необходимых для рисования объектов. Должен передаваться 4-м параметром в конструктор LWidget.

Пример:

LGraphics::LBaseComponent c;
auto obj = new LGraphics::LIRectangle(&app, nullptr, "image.png",&c);
auto b = new LGraphics::LIButton(&app, nullptr, "image.png",&c);
