Detailed Description

Класс приложения Lizard Graphics.

Объект этого класса должен объявляться перед любым из элементом LGraphics, т.к. LApp инициализурент OpenGL, GLFW окно и некоторые другие вспомогательные классы

void LGraphics::LApp::loop ( )

Бесконечный цикл

Открывает окно, при этом запуская бесконечный цикл, в котором рисуются все объекты сцены. Выход из цикла осуществляется при закрытии окна.
