Detailed Description

Класс приложения Lizard Graphics.

Объект этого класса должен объявляться перед любым из элементом LGraphics, т.к. LApp инициализурент OpenGL, GLFW окно и некоторые другие вспомогательные классы

Бесконечный цикл
