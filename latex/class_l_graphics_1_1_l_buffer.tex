\doxysubsection{Detailed Description}
Абстрактный класс буфера вершин. 

\mbox{\hyperlink{class_l_graphics_1_1_l_buffer}{L\+Buffer}} хранит в себе массив вершин объекта, массив текстурных координат, массив индексов (в каком порядке будут соединяться вершины при отрисовке).
