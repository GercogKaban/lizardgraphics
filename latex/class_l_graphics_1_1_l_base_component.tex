\doxysubsection{Detailed Description}
Компонент необходимы для конструкции виджетов Lizard Graphics. 

Сожержит в себе шейдер и буфер вершин, необходимых для рисования объектов. Должен передаваться 4-\/м параметром в конструктор L\+Widget.
